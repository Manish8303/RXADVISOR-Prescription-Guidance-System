What is RxAdvisor- Prescription Guidance &  Decision Support System?

RxAdvisor- Prescription Guidance &  Decision Support System is a machine learning-powered diagnostic assistant designed for healthcare professionals and researchers. It analyzes patient symptoms, age, and gender to:

  • Predict the most likely disease
  • Recommend the standard first-line medication
  • Provide model confidence score
  • Include clinical notes and caveats

Technology Stack

Python

Scikit-Learn

Flask API

Tailwind CSS

Trained on curated medical datasets using Random Forest Classifier with symptom vectorization.
